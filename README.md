# car-damage-detection

# Detección y localización de daños visibles en carrocería de vehículos mediante modelos de detección de objetos basados en YOLO

Este proyecto aborda la detección y localización automática de daños visibles en la carrocería de vehículos mediante modelos de detección de objetos basados en YOLOv8.

## Dataset

El dataset no está incluido en el repositorio por limitaciones de tamaño.
Se puede descargar en: https://universe.roboflow.com/car-damage-kadad/car-damage-v5/dataset/1

## Ejecución del notebook

Este notebook está optimizado para una ejecución directamente en Google Colab, subiéndolo a través de la pestaña de Github (Upload notebook > Github > pegar el enlace de este repositorio).

Instrucciones para la descarga de los datos: ejecutar las 5 primeras celdas del notebook
1. Clonar el repositorio (celda 1)
2. Cambiar de directorio (celda 2)
3. Instalar los requirements.txt (celda 3)
4. Descargar el dataset (celda 4)
5. Descargar los modelos (celda 5)

Se recomienda ejecutar estas primeras celdas individualmente, pero se puede ejecutar directamente todo el notebook con Run all.
